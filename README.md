# Getting Started with React Router

React Router 是完整的 React 路由解决方案

React Router 保持 UI 与 URL 同步。它拥有简单的 API 与强大的功能，例如代码延迟加载、动态路由匹配、以及帮助建立正确的位置过渡处理。
